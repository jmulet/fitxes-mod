Mòdul de Fitxes-Tutoria

El mòdul de fitxes-tutoria implementa les següents funcionalitats:

  • Cerca d'alumnes
    • Fitxa de dades personals
    • Fitxa dels cursos acadèmics
  • Obtenir informes
    • Llista de contrasenyes dels alumnes
    • Resum de les fitxes de tutoria
    • Orla dels grups
    • Llista i resum d'incidències SGD
  • Consultar/Iniciar actuacions de tutoria
    • Actuacions d'assistència-puntualitat
    • Actuacions de convivència
  • Preparar una nova entrevista amb la família
    • Sol·licituds d'informació a l'equip docent
    • Obtenir informació d'emergència
  • Justificar faltes d'assistència i retards*

*Requereix l'activació del plugin
