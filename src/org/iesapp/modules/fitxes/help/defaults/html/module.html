Tutorial forms module

This module implements the following features:

  • Student Search
    • Personal-information form
    • Academic-year form
  • Get reports
    • List of students' passwords
    • Summary of the tutorial forms
    • Get thumbnails of students
    • Get SGD incidences reports 
  • Start/See new tutorial actions
    • Actions of attendance and punctuality
    • Actions of behavior
  • Prepare a new interview with families
    • Ask information to teachers
    • Obtain emergency information
  • Justify absences and punctuality*

*Requires a plugin activation.
